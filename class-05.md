## Chapter11: _Color_

```
CSS properties
    opacity:    

    color:
    background-color:

Value examples
        0.5; *(opacity value between 0.0 and 1.0)*

        #ee3e80; *(HEX code)*
        DarkCyan; *(1 of 147 predefined color names)
        rgb(100,100,90);
        rgba(100,100,90,0.5); *(includes opacity value)*
        hsl(0,100%,100%);
        hsla(0,100%,100%,0.5); *(includes opacity value)*
```
